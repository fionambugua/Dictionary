# Wordly - Single Page Dictionary Application

Wordly is a Single Page Application (SPA) built using **HTML, CSS, and JavaScript** that allows users to search for English words and retrieve their meanings from the Free Dictionary API. The application updates dynamically without reloading the page, providing a smooth and interactive user experience.




## Technologies Used

- HTML5
- CSS3
- JavaScript
- Fetch API
- Free Dictionary API

## Installation

1. Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/Wordly.git
```

2. Navigate into the project folder.

```bash
cd Wordly
```

3. Open the project in Visual Studio Code.

4. Open `index.html` in your browser or use the Live Server extension.

---

## How to Use

1. Open the application.
2. Enter an English word into the search box.
3. Click the **Search** button.
4. View the word's:
   - Pronunciation
   - Definition
   - Part of Speech
   - Example
   - Synonyms
   - Audio pronunciation

If the word cannot be found, an error message will be displayed.
